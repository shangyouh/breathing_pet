# Breathing Pet 🐾

> 一只安静坐在桌面角落、按你舒服的节奏呼吸的小宠物。
> 它不会提醒你、不会弹窗、不会打扰——它只是呼吸，陪你工作。

![demo](demo.gif)

## 为什么有这个项目

研究有个名词叫 **Screen Apnea**——盯着屏幕工作时，人的呼吸会不自觉变浅、变急，有时甚至完全屏住。约 80% 的电脑工作者都有这个症状，但绝大多数人毫无察觉。

我自己写代码、开会、改方案的时候，总在某个瞬间突然意识到：胸口闷、肩膀紧、呼吸又浅又急。我想要一个**不打扰但能让我"想起呼吸"**的小东西。

所以做了这只小宠物。它住在屏幕角落，每 5 秒呼吸一次（吸气 2 秒，呼气 3 秒）。看着它的呼吸节奏，你的呼吸会不知不觉跟上——这是被动的、温柔的引导。

需要主动训练时，三击它即可进入 4-4-6-2 引导式呼吸训练。

## 特性

- 🌿 **自然呼吸节奏** — 吸气 2 秒（ease-in） + 呼气 3 秒（ease-out），不对称缓动模拟真实呼吸
- 🧘 **引导式呼吸训练** — 三击进入：吸气 4s → 屏息 4s → 呼气 6s → 暂停 2s，共 6 轮
- 🎨 **15 种 emoji 宠物 + 8 种莫兰迪色调** — 自动轮换或手动切换
- 👻 **完全被动** — 不弹窗、不通知、不打扰
- 🐧 **跨平台** — Windows / macOS / Linux
- 🪶 **零依赖** — 只用 Python 标准库

## 环境要求

- Python 3.8+
- Linux 用户需要安装 tkinter：`sudo apt install python3-tk`（其他系统自带）

## 快速开始

```bash
git clone https://github.com/YOUR_USERNAME/breathing-pet.git
cd breathing-pet
python breathing_pet.py
```

或者用启动脚本：

- **macOS / Linux**：`./breathing_pet.sh`
- **Windows**：双击 `breathing_pet.bat`

首次运行会自动在同目录生成 `config.json`，可编辑后重启自定义。

## 操作方式

| 操作 | 效果 |
|------|------|
| 拖拽 | 移动宠物位置（自动记忆） |
| 双击 | 切换到下一只宠物 |
| 三击 | 进入 / 退出呼吸训练模式 |
| 右键 | 打开菜单（换宠物 / 换颜色 / 退出） |

## 自定义配置

编辑 `config.json`：

```json
{
  "pet": {
    "skin_type": "emoji",
    "emoji_list": ["🐱", "🐶", "🐼", "🦄", "🐧"],
    "skin_rotate_minutes": 30,
    "window_size": [120, 120],
    "emoji_font_size": 64,
    "breath_inhale_ms": 2000,
    "breath_exhale_ms": 3000,
    "pet_color": "莫兰迪粉",
    "position": [100, 100]
  },
  "bubble": {
    "duration_seconds": 8,
    "font_size": 11,
    "max_width": 200
  }
}
```

### 可选颜色

`纯白` / `莫兰迪灰` / `莫兰迪粉` / `鼠尾草绿` / `薰衣草紫` / `雾蓝` / `奶油黄` / `淡灰绿`

### 调节呼吸节奏

研究显示**呼气长于吸气**更能激活副交感神经、降低焦虑：

| 用途 | 吸气 | 呼气 | 频率 |
|------|------|------|------|
| 默认陪伴 | 2s | 3s | 12 次/分钟 |
| 中度放松 | 4s | 6s | 6 次/分钟 |
| 深度放松 | 5s | 7s | 5 次/分钟 |

修改 `breath_inhale_ms` 和 `breath_exhale_ms` 即可。

## 呼吸训练模式

三击宠物进入：

```
  吸气 4s ──→ 屏息 4s ──→ 呼气 6s ──→ 暂停 2s
  100%→140%   保持 140%   140%→100%   保持 100%
   淡蓝色调    淡蓝色调      正常色      正常色

                  × 6 轮（共约 96 秒）
```

训练中再次点击宠物可提前结束。完成后回到普通陪伴模式。
